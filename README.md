# PDP-11_Compiler

Компилятор программ, написанных на языке ассемблера, переводящий их в байт-код и генерирующий листинг. 
Эмулирующет компиляцию программ на компьютере PDP-11.

## Установка
```bash
# 1. Клонировать репозиторий
git clone https://github.com/NikitaErokhov/PDP-11_Compiler.git

# 2. Перейти в директорию проекта
cd PDP-11_Compiler

# 3. Установить зависимости
pip install -r requirements.txt
```

## Использование
```bash
# 1. Стандартное использование
# --- Откомпилирует filename и создаст файлы filename.l и filename.o рядом с filename
python pdp11_compiler.py \<filename\> 

# 2. Глобальная компиляция - скрипт all_compile.sh
# --- Откомпилирует все файлы .pdp, находящиеся в pdp11_tests (формат хранения соотв.)
./all_compile.sh

# 3. Глобальная проверка - скрипт check.sh
# --- Сверит все файлы (*).pdp.l и (*).pdp.o, находящиеся в pdp11_tests (формат хранения соотв.),
#     с заранее заготовленными файлами exp_\1.pdp.l и exp_\2.pdp.o
./check.sh
```
